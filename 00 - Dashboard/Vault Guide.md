## Key Principles

### 1. Use [[Wikilinks]] Everywhere
- Link related notes using `[[Note Name]]`
- This builds your knowledge graph and makes connections visible
- The more links, the more useful the vault becomes

### 2. MOCs Are Your Navigation
- Start from [[Home]] for the dashboard
- Use Maps of Content (MOCs) to find topic-specific knowledge
- Each MOC is a curated index of related notes

### 3. Daily Notes Are Your Capture Point
- Start each day with a Daily Note (use the template)
- Capture thoughts, tasks, and learnings here
- Process items weekly into permanent notes

### 4. PARA Method
This vault follows the PARA method:
- **P**rojects — Active work with deadlines
- **A**reas — Ongoing responsibilities
- **R**esources — Reference material
- **A**rchive — Completed items

### 5. Atomic Notes
- Each note should cover ONE concept or topic
- Notes should be self-contained but linked to related notes
- Use tags and frontmatter for organization
