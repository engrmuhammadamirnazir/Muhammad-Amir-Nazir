# PRJ — Aligners Pakistan

> Custom Odoo store development for dental aligner manufacturing and distribution.

---

## Overview

Development of an integrated Odoo-based system for Aligners Pakistan, covering the full workflow from patient data management through scan ordering, invoicing, and setup designing.

---

## Requirements

### 1. Patient Bio Data
- Patient registration: name, telephone, clinic/hospital, patient number, email
- Upload functionality: STL files, pictures, X-rays, documents
- Internal file tree storage system for organized document management
- Doctor messaging portal with internal commenting system

### 2. Scan Order Management
- Scan date tracking, completion status, and payment
- Doctor name and scan location
- File upload into organized storage system
- Preference for OneDrive integration (optional)

### 3. Invoicing
- Patient name, scan date, discount and tax columns
- Patient reference number, invoice number
- Paid/unpaid status tracking

### 4. Setup Designing
- Assigned doctor and case tracking
- Date of case received, delivery date, and due date
- File management for setup files, pictures, and videos

---

## Technical Notes

- Built on Odoo ERP platform
- Custom module development required
- Integration with file storage (OneDrive preferred)
- Patient portal and doctor messaging features
