# Odoo ERP Overview

> Canonical reference for understanding Odoo ERP — architecture, modules, and ecosystem.

---

## What is Odoo?

Odoo is an open-source ERP (Enterprise Resource Planning) platform offering integrated business applications for CRM, sales, inventory, accounting, manufacturing, HR, and project management. It uses a modular architecture where businesses install only the apps they need.

---

## Technical Architecture

| Layer | Technology |
|-------|-----------|
| **Backend** | Python |
| **Frontend** | JavaScript / Owl framework |
| **Database** | PostgreSQL |
| **API** | XML-RPC, JSON-RPC |
| **Deployment** | On-premise, cloud (Odoo.sh), Docker |

## Editions

### Community Edition (Free & Open Source)
- Core business apps
- Limited features per module
- Community support

### Enterprise Edition (Paid)
- Full feature set
- Studio (no-code customization)
- IoT, Barcode, Helpdesk, Appraisal
- Official support and hosting

> See also: [[Odoo Community vs Enterprise]] for detailed comparison
